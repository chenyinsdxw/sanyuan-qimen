## 一、核心原则

### 1. 用神第一性

同一个盘，问工作看开门，问合作看六合，问婚恋看乙庚+六合。**不同问题有不同用神体系，不能一刀切。**

### 2. 符号优先级

一个宫位里有多个符号，按此顺序读取：
**九星（行业/性质）→ 八门（状态/人事）→ 八神（氛围/暗能量）→ 格局（天地气运交合）**

例：年命宫天辅+伤门+白虎+庚戊
- 天辅（文化教育）→ 行业性质
- 伤门（压力/争斗）→ 工作状态
- 白虎（权威/严格）→ 氛围
- 庚戊（飞宫格→变动不顺）→ 当前气运

### 3. 年命双维度

年干看宫位，年支也看宫位（六十甲子遁九宫查年支位置）。二者同宫则信号放大，不同宫则分看各自的状态。**不能只看年干。**

### 4. 知识体系 > 案例积累

每个实战案例用于检验理论理解，不存入技能。纠偏结论反哺到知识结构中。

### 5. 符号组合优先于单一生克

五行生克不是唯一法则。年干辛离火生日干癸艮土，看似相生，但实际母亲态度强势——**符号的全局组合（天辅庚+杜门丁+玄武）传达的信息比简单的五行生克更优先。** 判断时，先看宫位内的符号组合，再用生克关系做辅助印证。

### 6. 数象连接

九宫数不只是宫位编号，是事物量级的指示：
- 戊（财）+ 宫位数 = 金额级别（戊在坤2 → 2万/2千）
- 年命宫的死门/杜门/景门都有时空量感
- 应期推断中宫位数的直接换算

### 7. 伏吟局的两层信息

伏吟局不只是一张"当前状态"的快照，它记录的是**长期积累的势能**：
- **历史积累层**：星门在原位，代表这个领域有长期沉淀——包括成就、能力、经验
- **当前卡点层**：年命宫的四害（空亡/入墓/击刑/门迫）代表**当前被卡住的地方**，不代表全部历史

× 错误解读："年命空亡 = 他一直没工作"
✓ 正确解读："年命空亡 = 他**现在**卡住了，空亡之前的轨迹要另看"

实战中先用用户背景确认"状态反转点"在哪，再决定符号解哪一面。

## 三、分析流程

核心思路：**问题→用神→辐射分析。** 不是固定流程，而是从问题出发自然展开。

### Step 1：定位用神

听问题→确定问题类型→找出该类型对应的核心用神（见第二章）。

例：问工作 → 开门。问合作 → 六合。问能不能做成 → 时干+生门。

### Step 2：看用神宫的核心状态

- 空亡？→ 吉事谨防变数，凶事可暂缓
- 入墓？→ 能量被封存，吉神有心无力
- 马星？→ 有动的趋势（实马则动真格，空马则折腾不落地）
- 格局（十干克应）？→ 天地气运交合的状态

### Step 3：看用神与人的关系

- 与日干/年命的生克？→ 事体对求测人的影响
- 与日干同宫？→ 人与事深度绑定
- 日干临值符？→ 有大领导/贵人看中

### Step 4：辐射关联用神

每个问题类型都有多维度用神（见第二章），逐一查看：
- 工作：值符（大老板）→ 年干（直属领导）→ 月干（同事竞争）
- 合作：生门（利润）→ 戊（资金）→ 时干（对方）
- 婚恋：乙（女）→ 庚（男）→ 六合（婚姻）→ 癸（情网）

### Step 5：三元能量流（必要时）

当问题涉及"能否走通""是否长久"时，看三元能量路线：

**元神（过去/根基）→ 值符（现在/气运核心）→ 天乙（未来/结果）**

- 一气贯通（纯生或纯克）→ 能量顺畅（生=牵引，克=驱动，都是推力）
- 比和 → 能量不流通
- 值符空亡 → 能量全漏，天乙再吉也虚
- 值符入墓 → 能量被封
- 三元归一（值符=天乙）→ 时间线压缩，发展空间有限

**元神为根，天乙为果，值符为干。根果同处则事有终始，干空则根果皆虚。**

### Step 6：定应期

- 空亡填实/冲实之时（检查填实后引动什么凶象）
- 马星发动之时
- 格局中天干逢冲合之时
- 伏吟局：日支填实马星则当天应（坤2=不超过2天）
- 反吟速（日/时），伏吟缓（月/年）

## 四、四害详解（空墓刑迫）

四害是奇门断局的核心切入点，矛盾所在即信息所在。**先定"有没有"（空亡），再定"好不好"（格局）。**

### 4.1 空亡

**吉门逢空亡：** 发挥不了吉
**凶门逢空亡：** 发挥不了凶，但引动时更凶

**空亡的体现：** 未发生、变化、转移、暂停、消失、死亡、看不见、被遮挡、有漏洞、不安定

**空亡的转移机制：**
- **对宫转移** — 如照镜子，自己看不到自己的脸→镜子里的脸是空亡对宫
- **先后天通窍转移** — 离宫空亡与乾宫通窍，相通的宫位有信息流通

**空亡遇马星：** 可以"动空"，但力量不强，如同空中的马/坑里的马——在局限中折腾

**六种补空法：**
1. **冲空** — 六冲地支来冲旬空
2. **填空** — 遇到旬空的本支
3. **脱空** — 脱离当旬
4. **动空** — 空亡宫遇马星等动象
5. **旺空** — 空亡宫在旺月旺时
6. **出空** — 时间走出空亡旬

### 4.2 入墓

| 类型 | 表现 |
|------|------|
| **吉神入墓** | 有心无力，好事难成 |
| **凶神入墓** | 徒有其表难害，凶性被封锁 |
| **格局入墓** | 吉凶能量受抑制 |

**入墓与入库的区别：** 入墓=真正埋葬困顿；入库=暂时存放蓄力。

**八门入墓：** 休门/生门/死门入巽宫（辰）；伤门/杜门入坤宫（未）；惊门/开门入艮宫（丑）；景门入乾宫（戌）

### 4.3 击刑

六仪击刑加重规则：**当该仪为旬首时更凶。**

- 甲子戊落震宫（子卯刑）→ 无礼之刑
- 甲戌己落坤宫（戌未刑）/ 甲申庚落艮宫（申寅刑）/ 甲午辛落离宫（午自刑）
- 甲辰壬落巽宫（辰自刑）/ 甲寅癸落巽宫（寅巳刑）

### 4.4 门宫迫

**门迫 = 门克宫**（人为主体不适应环境）：**吉门迫则吉不显，凶门迫则更凶。**
**门制（宫迫）= 宫克门**（环境压迫人）。

```
门迫：从里向外打开窗   → 人为主，不适应单位
门制：从外向里打开窗   → 单位压迫人，被辞退
```

## 六、应期推断（7种方法）

1. **四害应期** — 空墓刑迫填实冲空之时
2. **十干格局应期** — 格局中天干逢冲、合之时
3. **落宫旺衰** — 用神落宫旺衰 + 地支
4. **大局定快慢** — 反吟速（日/时），伏吟缓（月/年）
5. **特殊格局定** — 如青龙逃走见马星动
6. **值使应期** — 值使宫能量到位
7. **能量应期** — 五行能量最旺/最衰之时

### 值符空亡局的应期陷阱

值符空亡局，即使到了填实的应期，也要检查填实后**引动了什么凶象**：
- 填实后引动死门 → 可能有官非/调整不成
- 填实后引动天芮 → 可能有健康/纠纷问题
- 填实后引动击刑 → 引火烧身

**"即使沾边，也会引火烧身" → 应到了不代表好事。填实空亡前先评估被动符号的吉凶。**

### 伏吟局应期特例

伏吟局中值符宫有马星、且当天日支正好是马星所在的地支时，称为**日支填实马星**。若当天没出，取落宫数定上限（坤2=不超过2天）。

## 九、心法八式（核心方法论）

严格按此顺序拆解任意盘面，作为"问题→用神"方法论的深度补充：

```
1. 元气（观大局卦气）
   └─ 值符宫状态（空亡/入墓/马星/反吟伏吟）
       └─ 定性事件基调

2. 起元（定元神宫）
   └─ 直符星的原始宫位（根基、本家、后勤）
       └─ 元神是否与值符重合？旺衰如何？

3. 符使（定值符值使）
   ├─ 值符（直符）→ 事件核心气运，不等于"人"
   ├─ 值使（直使）→ 执行力、行动力
   └─ 符使的生克关系 → 气运与执行力的配合度

4. 天乙（定结果趋势）
   └─ 时干宫（天乙宫）→ 事体结果
       └─ 是否与元神重合？→ 好事坏事都放大

5. 飞伏（星门动静）
   ├─ 九星伏吟/反吟 → 气运的动静
   ├─ 八门伏吟/反吟 → 人事的动静
   └─ 星门伏吟的两层信息：历史积累层 ≠ 当前卡点层

6. 递进（三元能量流）
   ├─ 元神→值符→天乙 能量走向
   ├─ 生为牵引（顺境），克为驱动（逆境），空亡为泄气（变数）
   └─ 关键：值符宫是否空亡？→ 空则能量全漏

7. 动静（查四害焦点）
   ├─ 空亡 → 填实/冲实/马星触发
   ├─ 马星 → 实马还是空马？
   ├─ 门迫/击刑/入墓 → 哪里被卡住
   └─ 先定"有没有"（空亡），再定"好不好"（格局）

8. 命主（归年命日干 + 运筹建议）
   ├─ 年命宫 → 被问之人的状态
   ├─ 日干宫 → 问卦人自身
   └─ 择时/择方/行为策略
```

**核心口诀：**
> "值符最怕飞伏空，泊空亡就没有发动力。"
> "符者令，使者行。令行禁止，令在前，行在后。"
> "直使是踩油门，直使克元神就是挂倒挡踩油门。"
> "空亡=待机，入墓=死机。"
> "生和克都是推动力，比和没有气运流通。"
> "伏吟局，静中寻动，动处便是应期。"
> "天显时格罪赦免，马星一动破伏吟。"
> "退吉进凶不强求，守住现位也正途。"

### 10.2 九遁格局

核心：**吉门+三奇+吉神**之间的配合（神遁：丙+九天+生门；天遁：乙+生门+丙等）

### 10.3 五凶格

| 格局 | 组合 | 含义 |
|------|------|------|
| 青龙逃走 | 乙+辛 | 吉变凶，逃遁 |
| 白虎猖狂 | 辛+乙 | 凶势加剧 |
| 朱雀投江 | 丁+癸 | 文书遗失/声名受损 |
| 腾蛇夭矫 | 癸+丁 | 虚惊缠绕 |
| 荧入白/白入荧 | 丙+庚/庚+丙 | 火入金/金入火，主消亡 |

### 10.4 其他重要格局

- **大格：** 庚+癸 → 大事被破坏
- **小格（上格）：** 庚+壬 → 移动/变动反复
- **刑格：** 庚+己 → 纠纷/官司
- **伏宫格：** 庚临值符本人；**飞宫格：** 值符临庚
- **时干入墓：** 时干入墓则事难成
- **悖格：** 丙+乙/丙+丙 → 事情反常
- **天网四张：** 癸临四宫 → 困境难脱
- **五不遇时：** 时干克日干 → 事难成/多挫败

### 10.5 寄宫思维

禽星值符时"符无真家，门无真使"——借势、借力打力。用神遇禽星寄宫，往往是借助别人力量、中介、渠道。

**寄宫伏吟：** 身不动心在动，暗藏杀机，战略欺骗。

### 10.6 通窍法则

空亡在先后天的转移。先后天通窍犹如一个竹筒，一头是空亡，一头是后天。通窍也就是相通，有信息流通。

**通窍生效的两大前提（缺一不可）：**
1. **马星引动** — 空亡宫的先后天宫位遇马星
2. **格局冲** — 如宫位内的天干冲格

**先天飞后天（顺水推舟）：** 主已发生之事
**后天飞先天（逆水行舟）：** 主隐秘因果或生死剧变

## 十一、修习次第与避忌

### 修习路径

1. **入门** — 理解奇门是什么→掌握符号体系（八门/九星/八神/八卦/十干/十二地支）
2. **基础** — 排盘10步→万物类象→用神选择→分类占断
3. **进阶** — 四害（空墓刑迫）→符使路线→三元递进→特殊格局
4. **高阶** — 心法八式→三元换用→大运流年→运筹克应
5. **实战** — 案例对照→纠偏积累→体系自洽

### 断卦要点

- **直使克元神 = 挂倒挡踩油门** — 执行力与根基方向相反
- **反吟看用神吉凶** — 主返回（人回来或灵魂回来）
- **时干人墓** — 入时支墓+入宫位墓=双重入墓加重
- **震巽相邻同属木** — 认识的人、有交集
- **戊+开门=财务工作** — 不只是资金，直指财务/金融岗位
- **螣蛇+空亡=承诺不兑现** — 空亡主虚，螣蛇主缠绕

### 戒忌

1. 信息不足时**明确指出待验证点**，不强行圆局
2. 不要用生克关系覆盖符号组合的判断
3. 先用用户背景确认"状态反转点"，再决定符号解哪一面
4. 值符空亡局，填实不等于好事——先评估被引动符号的吉凶

1. 分析从"问题→用神"开始，不预设流程步骤
2. 符号组合靠推导不靠记忆——按"星→门→神→格局"的顺序读一个宫

6. 先定"有没有"（空亡），再定"好不好"（格局）
